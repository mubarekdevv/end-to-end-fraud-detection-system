# End-to-End Machine Learning System for Financial Fraud Detection and Risk Scoring

## Overview

This project is a production-style machine learning system designed to detect fraudulent financial transactions and assign transaction risk scores in real time.

The system demonstrates the complete machine learning lifecycle including:

- Data preprocessing
- Exploratory data analysis
- Feature engineering
- Model training
- Fraud prediction
- Risk scoring
- API deployment
- Model evaluation
- End-to-end system architecture

## Objectives

- Detect fraudulent transactions using machine learning
- Generate transaction risk scores
- Build an explainable AI pipeline
- Simulate a real-world fintech fraud detection system
- Practice production-ready ML engineering workflows
